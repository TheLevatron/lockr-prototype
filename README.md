# LockR Modern UI

This branch introduces a modern React + Tailwind UI scaffold and applies consistent spacing, line-height, and margins across all pages.

## Getting Started

- npm install
- npm run dev

## Spacing & Vertical Rhythm

- Body line-height: 1.7; headings: 1.3
- Section spacing: `.section` uses `space-y-5`
- Cards: `p-6` and `space-y-4`
- Modals/Drawers: header/footer `p-5`; content `p-6`
- Grids: gaps standardized to 4–5

## Pages

- User: Splash, Login, Floor Plan, Locker Grid, Reservation Drawer, Verified
- Admin: Wizard, Main, Floor Editor, Locker Update
